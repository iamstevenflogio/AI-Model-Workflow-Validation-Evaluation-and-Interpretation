# About Dataset:

This project uses The Detailed Bridge Inventory dataset through the PH Barangay Downloader, a centralized geospatial data platform developed by Zenterra Systems Solutions. 
https://phadmindownloader.vercel.app/

This inventory captures comprehensive details on bridges across the national road network,
including structural type (concrete, steel, or timber), current condition rating, span length,
carriageway width, load capacity limits, and year of construction or last major rehabilitation.

This project was done to demonstrate AI model workflow, validation, evaluation, and interpretation for my final Data Science 4 project.
